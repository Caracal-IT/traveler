# Finding Things

Finding things on the command line is a common task. There are several commands that can help you find files,   
directories, and even text within files. Here are some of the most commonly used commands for finding things   
on the command line:

## Basic Commands

- `find`: Searches for files and directories based on various criteria.
- `locate`: Uses a database to quickly find files by name.
- `grep`: Searches for text within files.
- `findstr`: Searches for text within files (Windows).

### Locate

The `locate` command is used to quickly find files by name. It searches a database of files and directories   
that have been indexed by the `updatedb` command. The `locate` command is much faster than the `find` command   
because it uses a pre-built database. However, the `locate` command only works with files that have been indexed   
by the `updatedb` command, which may not include all files on the system.

On modern Ubuntu systems, you can install it using:

```bash
$ sudo apt update
$ sudo apt install locate
$ sudo updatedb
```

Example usage:

```bash
$ locate filename
$ locate *.txt
```

#### Parameters

| Name     |          | Description                         |
|----------|----------|-------------------------------------|
| filename | Required | The name of the file to search for. |
| *.txt    | Optional | Search for all text files.          |
| -i       | Optional | Case-insensitive search.            |
| -r       | Optional | Search subdirectories.              |
| -m       | Optional | Search for multiple matches.        |
| -n       | Optional | Show line numbers.                  |
| -x       | Optional | Show file extensions.               |
| -0       | Optional | Use NUL as the file separator.      |
