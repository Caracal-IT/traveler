# Piping

Piping in Linux command line allows you to connect the output of one command to the input of another,   
creating a chain of commands that can be executed sequentially. This is done using the pipe symbol (`|`),   
which redirects the standard output (stdout) of the preceding command to the standard input (stdin)   
of the following command.


```bash
$ command1 | command2
$ date | rev
$ ls -l /usr/bin | less
$ ls -l /usr/bin -1 | wc -l
```

## tr

The tr command is used to convert characters in a file. It can be used to replace characters, delete characters,   
or translate characters in a file. It is commonly used for text processing tasks such as removing special characters,   
converting case, and formatting text. For example, to convert all lowercase letters to uppercase in a file, you can   
use the following command:

```bash
$ tr '[:lower:]' '[:upper:]' < input.txt > output.txt
$ cat msg.txt | tr s S
```

This command reads the contents of `input.txt`, converts all lowercase letters to uppercase, and writes the   
result to `output.txt`.

### Character classes

The tr command uses character classes to specify the characters to be converted.

| Class      | Description                               |
|------------|-------------------------------------------|
| [:lower:]  | Lowercase letters                         |
| [:upper:]  | Uppercase letters                         |
| [:digit:]  | Digits                                    |
| [:punct:]  | Punctuation                               |
| [:space:]  | Whitespace characters                     |
| [:alnum:]  | Alphanumeric characters                   |
| [:alpha:]  | Alphabetic characters                     |
| [:graph:]  | Printable characters excluding whitespace |
| [:print:]  | Printable characters including whitespace |
| [:xdigit:] | Hexadecimal digits                        |
| [:blank:]  | Whitespace characters                     |
| [:cntrl:]  | Control characters                        |

## tee

The tee command is used to copy the input to both stdout and a file. It is useful for capturing output from a command   
and saving it to a file simultaneously. For example, to copy the output of a command to both stdout and a file,   
you can use the following command:

```bash
$ command | tee output.txt
$ cat colours.txt words.txt | tee output.txt | wc -l
```

This command runs the `command`, and its output is both displayed on the terminal and saved to `output.txt`.  
It also counts the number of lines in the output and displays the result.
