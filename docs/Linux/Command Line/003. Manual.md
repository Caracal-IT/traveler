# Manual

The manual command. It provides detailed information about commands and their usage.   
Type q to quit. Scroll using the arrow keys. Type / to search.  

```bash
man command
man section command
```

**Example:**

```bash
man ncal
```

**Searching the Manual:**

```bash
man -k keyword
man -K passwd
```
