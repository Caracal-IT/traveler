# Command Line Basics

## Command Structure

Commands in the Linux command line typically follow a specific structure. The basic format is:

```bash
command -options arguments
```

- **Command**: The main action to be performed.
- **Options**: Optional settings that modify the command's behavior.
- **Arguments**: Data or files on which the command operates.

## clear

```bash
clear
```

The clear command is used to clear the terminal screen.   
The output is: `No output`.

## date

```bash
date
```

The date command is used to display the current date and time.  
The output is: `Thu Jan  1 00:00:00 UTC 1970`. 

## ncal (cal)

```bash
ncal
```

The ncal (new cal) command is used to display a calendar.  
The output is: 
```bash
    January 2026      
Su     4 11 18 25   
Mo     5 12 19 26   
Tu     6 13 20 27   
We     7 14 21 28   
Th  1  8 15 22 29   
Fr  2  9 16 23 30   
Sa  3 10 17 24 31 
```
