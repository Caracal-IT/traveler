# Commands

This section contains a list of useful Linux command line commands.

1. clear - clears the screen
2. echo - prints the arguments
3. date - prints the current date and time
4. ncal - prints the calendar
5. man - displays the manual page for a command
6. help - displays the manual page for a command, help cd
7. type - displays the type of command. cmd, built-in shell command, shell function, or alias
8. which - displays the path of a command
9. whoami – prints the username of the current user
10. who - prints the users currently logged in
11. pwd - prints the current working directory
12. cd - changes the current working directory
13. ls - lists the contents of the current directory 
14. touch – creates a new file or updates the access and modification times of an existing file
15. nano - opens the default text editor
16. cat - concatenates and prints the files
17. rm – removes files or directories
18. mkdir – creates a new directory
19. mkfifo - creates a named pipe (FIFO)
20. rmdir – removes an empty directory
21. cp - copies files or directories
22. mv - moves or renames files or directories
23. find - searches for files and directories
24. grep - searches for a pattern in files
25. chmod - changes file permissions
26. chown - changes file ownership
27. sudo - runs a command with elevated privileges
28. file - determines the file type
