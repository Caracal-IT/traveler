# Nano

Nano is a user-friendly and resource-efficient text editor, popular among developers and 
system administrators for its straightforwardness and speed. Its appeal to developers 
and system administrators stems from its uncomplicated design and high performance.

## Basics

To open a file in Nano, use the following command:

```bash
nano filename
```

Press Ctrl-G to display the help menu, which provides a list of available commands and their functions.

## Useful Shortcuts

|  Shortcut  |           Function            |
|:----------:|:-----------------------------:|
|   Ctrl+G   |       Display help menu       |
|   Ctrl+O   |     Save file (WriteOut)      |
|   Ctrl+X   |           Exit Nano           |
|   Ctrl+R   |       Read/Insert file        |
|   Ctrl+W   |        Search for text        |
|   Ctrl+\   |      Search and replace       |
|   Alt+W    |    Search next occurrence     |
|   Ctrl+K   |       Cut current line        |
|   Ctrl+U   |        Paste cut text         |
|   Ctrl+J   |       Justify paragraph       |
|   Ctrl+C   |     Show cursor position      |
|   Ctrl+_   | Go to line and column number  |
|   Alt+U    |       Undo last action        |
|   Alt+E    |    Redo last undone action    |
|   Ctrl+A   |   Move to beginning of line   |
|   Ctrl+E   |      Move to end of line      |
|   Ctrl+Y   |            Page up            |
|   Ctrl+V   |           Page down           |
|   Alt+\    |    Go to beginning of file    |
|   Alt+/    |       Go to end of file       |
| Ctrl+Space |     Move forward one word     |
| Alt+Space  |    Move backward one word     |
|   Alt+6    |       Copy current line       |
|   Ctrl+T   |  Spell check (if available)   |
|   Alt+3    |    Comment/Uncomment line     |
|   Alt+]    |    Complete bracket/quote     |
|   Ctrl+D   | Delete character under cursor |
| Alt+Delete |   Delete word to the right    |

## Configure Nano

To customize Nano, you can edit the configuration file located at `~/.nanorc` or `/etc/nanorc`.   
This file allows you to set various options and preferences for Nano.
