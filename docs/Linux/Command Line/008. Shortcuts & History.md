# Shortcuts & History

1. Use the up and down arrow keys to navigate through command history.
2. Press `Ctrl + R` to search through command history.
3. Use `Ctrl + A` to move the cursor to the beginning of the line.
4. Use `Ctrl + E` to move the cursor to the end of the line.
5. Use `Ctrl + L` to clear the screen.
6. Use `Ctrl + C` to interrupt a running command.
7. Use `Ctrl + D` to exit the current shell.
8. Use `Ctrl + Z` to suspend a running command.
9. Use `Ctrl + W` to delete the word before the cursor.
10. Use `Ctrl + U` to delete the line before the cursor.
11. Use `Ctrl + K` to delete the line after the cursor.
12. Use `history` to display the command history.
13. Use `Ctrl + Y` to paste the last deleted text.
14. Use `Ctrl + F` to move forward one character at a time.
15. Use `Ctrl + B` to move backward one character at a time.
16. Use `Alt + F` to move forward one word.
17. Use `Alt + B` to move backward one word.
18. Use `Ctrl + T` to swap the last two characters before the cursor.
